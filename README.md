# DBMS Rentals App

## Overview
This project is a course assignment for **Database Management Systems (DBMS)**.  
It implements a rentals management web application based on a relational database, focusing on correct schema design, SQL querying, and data analysis.

## System Description
The system manages information about:
- **Owners**
- **Apartments**
- **Rentals**

It supports querying, inserting, and analyzing rental data according to predefined business rules.

## Technologies
- **Python / Django** – application logic and server-side rendering  
- **Microsoft SQL Server (MSSQL)** – relational database  
- **SQL** – complex queries and views  
- **HTML / CSS** – user interface  

## Database Features
- Relational schema with primary and foreign keys
- SQL **VIEWs** for complex analytical queries
- Aggregations, filtering, and sorting
- Validation and constraint handling at the application level

## Project Structure
- `Rentals_App/` – Django application (views, models, routing)
- `templates/` – HTML pages rendered by Django
- `static/` – static assets (CSS, images)
- `queries_views.sql` – SQL VIEW definitions for DBMS queries
